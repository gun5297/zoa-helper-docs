🔒 개인정보처리방침

Zoa Agency Helper (조아 에이전시 헬퍼)

Zoa Agency Helper(이하 "본 확장 프로그램")는 사용자의 개인정보 보호를 매우 중요하게 생각하며, 관련 법령을 준수합니다. 본 개인정보처리방침은 사용자가 제공하는 정보가 어떻게 처리되는지 설명합니다.

1. 수집하는 개인정보

본 확장 프로그램은 팬이벤트 응모 자동기입 기능을 위해 다음 정보를 사용자의 로컬 기기에만 저장합니다:

  • 이름
  • 연락처 (전화번호)
  • 이메일 주소
  • 생년월일
  • SNS 계정 정보 (선택적)
  • 응모자 정보 (선택적)
⚠️ 중요: 모든 데이터는 사용자의 브라우저 로컬 스토리지(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다.

2. 개인정보의 이용 목적

수집된 정보는 오직 다음 목적으로만 사용됩니다:

  • K-POP 팬이벤트 응모 양식 자동 기입
  • 쇼핑몰 주문 양식 자동 완성

3. 개인정보의 보관 및 파기

  • 모든 정보는 사용자의 브라우저 로컬 저장소에만 보관됩니다.
  • 확장 프로그램을 삭제하면 모든 저장된 데이터가 자동으로 삭제됩니다.
  • 사용자는 언제든지 확장 프로그램 설정에서 저장된 정보를 직접 삭제할 수 있습니다.

4. 개인정보의 제3자 제공

본 확장 프로그램은 사용자의 개인정보를 제3자에게 제공하거나 판매하지 않습니다. 모든 데이터는 사용자의 기기에서만 처리됩니다.

5. 외부 서비스 연동

본 확장 프로그램은 Google Apps Script를 통해 쇼핑몰 설정 정보를 불러옵니다. 이 과정에서:

  • 사용자의 개인정보는 전송되지 않습니다.
  • 쇼핑몰 URL 매칭을 위한 설정 데이터만 수신합니다.

6. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 저장된 개인정보의 열람, 수정, 삭제
  • 확장 프로그램 삭제를 통한 모든 데이터 삭제
  • 개인정보 처리에 관한 문의

7. 보안

본 확장 프로그램은 사용자의 데이터 보안을 위해:

  • Chrome의 안전한 storage API를 사용합니다.
  • 외부 서버로 개인정보를 전송하지 않습니다.
  • 필요한 최소한의 권한만 요청합니다.

8. 개인정보처리방침의 변경

본 개인정보처리방침은 법령이나 서비스 변경에 따라 수정될 수 있습니다. 변경 시 확장 프로그램 업데이트 또는 본 페이지를 통해 공지합니다.
