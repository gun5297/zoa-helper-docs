🔒 개인정보처리방침

Zoa Agency Helper (조아 에이전시 헬퍼)
Chrome 확장 프로그램 ID: mblbincdcjcebiomcoipgpjeejpghekb

Zoa Agency Helper(이하 "본 확장 프로그램")는 사용자의 개인정보 보호를 매우 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 사용자가 제공하는 정보가 어떻게 수집, 처리, 보관, 공유되는지 상세히 설명합니다.

📌 핵심 요약: 본 확장 프로그램은 사용자가 입력한 정보를 오직 사용자의 로컬 기기에만 저장하며, 어떠한 외부 서버로도 개인정보를 전송하거나 수집하지 않습니다.

1. 수집하는 개인정보 항목

본 확장 프로그램은 팬이벤트 응모 및 쇼핑몰 주문 양식 자동 기입 기능을 위해 다음 정보를 수집합니다:

수집 항목 필수/선택 수집 목적
이름 선택 응모 양식 자동 기입
연락처 (전화번호) 선택 응모 양식 자동 기입
이메일 주소 선택 응모 양식 자동 기입
생년월일 선택 응모 양식 자동 기입
SNS 계정 정보 (X/Twitter, Instagram 등) 선택 응모 양식 자동 기입
주소 정보 선택 배송지 양식 자동 기입
⚠️ 중요: 위 모든 정보는 사용자가 직접 확장 프로그램 설정 화면에서 입력하며, 자동으로 수집되지 않습니다. 입력 여부는 전적으로 사용자의 선택입니다.

2. 개인정보 수집 방법

본 확장 프로그램은 다음과 같은 방법으로만 개인정보를 수집합니다:

  • 사용자 직접 입력: 확장 프로그램의 팝업 설정 화면에서 사용자가 직접 정보를 입력
  • 자동 수집 없음: 웹 브라우징 활동, 방문 기록, 쿠키 등을 자동으로 수집하지 않음
  • 백그라운드 수집 없음: 사용자 모르게 정보를 수집하는 기능 없음

3. 개인정보의 처리 및 이용 목적

수집된 정보는 오직 다음 목적으로만 처리됩니다:

  • 양식 자동 기입: K-POP 팬이벤트 응모 페이지에서 응모 양식을 자동으로 채움
  • 주문서 자동 완성: 쇼핑몰 주문 페이지에서 배송 정보를 자동으로 채움

위 목적 외에 다른 용도로 개인정보를 처리하지 않으며, 이용 목적이 변경되는 경우 사전에 동의를 구합니다.

4. 개인정보의 보관 위치 및 방법

4.1 보관 위치

  • 모든 개인정보는 사용자의 로컬 기기에만 저장됩니다.
  • Chrome 브라우저의 chrome.storage.local API를 사용하여 브라우저 내부에 암호화되어 저장됩니다.
  • 외부 서버 저장 없음: 개발자 서버, 클라우드 서비스, 제3자 서버 등 어떠한 외부 저장소에도 개인정보를 저장하지 않습니다.

4.2 보관 기간

  • 사용자가 직접 삭제하거나 확장 프로그램을 제거할 때까지 로컬에 보관됩니다.
  • 보관 기간에 제한이 없으며, 사용자가 원할 때 언제든지 즉시 삭제할 수 있습니다.

5. 개인정보의 파기 절차 및 방법

사용자는 다음 방법으로 저장된 개인정보를 완전히 파기할 수 있습니다:

  • 설정에서 삭제: 확장 프로그램 팝업 → 설정에서 저장된 정보를 개별 또는 전체 삭제
  • 확장 프로그램 제거: Chrome에서 확장 프로그램을 삭제하면 모든 저장 데이터가 자동으로 완전히 삭제됨
  • 브라우저 데이터 삭제: Chrome 설정에서 확장 프로그램 데이터를 삭제

파기된 개인정보는 복구가 불가능하며, 어떠한 백업도 보관하지 않습니다.

6. 개인정보의 제3자 제공 및 공유

본 확장 프로그램은 사용자의 개인정보를 제3자에게 제공, 공유, 판매하지 않습니다.
  • 광고 네트워크와 정보를 공유하지 않습니다.
  • 분석 서비스에 개인정보를 전송하지 않습니다.
  • 어떠한 외부 서비스와도 개인정보를 공유하지 않습니다.
  • 법적 요구가 있는 경우에도, 개발자는 사용자 기기에 저장된 정보에 접근할 수 없습니다.

7. 외부 네트워크 통신

본 확장 프로그램이 수행하는 유일한 외부 통신은 다음과 같습니다:

  • 쇼핑몰 설정 데이터 수신: Google Apps Script를 통해 쇼핑몰 URL 패턴 및 입력 필드 설정 정보를 받아옵니다.

이 과정에서:

  • 사용자의 개인정보는 절대 전송되지 않습니다.
  • 오직 쇼핑몰 설정 정보(URL 패턴, CSS 선택자 등)만 수신합니다.
  • 사용자를 식별할 수 있는 어떠한 정보도 전송하지 않습니다.

8. 사용자의 권리와 행사 방법

사용자는 개인정보 주체로서 다음과 같은 권리를 행사할 수 있습니다:

  • 열람권: 확장 프로그램 설정에서 저장된 모든 정보를 확인할 수 있습니다.
  • 정정권: 저장된 정보를 언제든지 수정할 수 있습니다.
  • 삭제권: 저장된 정보를 부분 또는 전체 삭제할 수 있습니다.
  • 처리정지권: 확장 프로그램을 비활성화하거나 제거하여 처리를 정지할 수 있습니다.

모든 권리는 확장 프로그램 내에서 즉시 행사할 수 있으며, 별도의 요청 절차가 필요하지 않습니다.

9. 개인정보의 안전성 확보 조치

본 확장 프로그램은 개인정보의 안전성 확보를 위해 다음 조치를 취하고 있습니다:

  • 로컬 저장: 모든 데이터를 사용자 기기에만 저장하여 외부 유출 위험 원천 차단
  • Chrome Storage API: Chrome에서 제공하는 안전한 저장소 API 사용
  • 최소 권한 원칙: 기능 수행에 필요한 최소한의 브라우저 권한만 요청
  • 외부 전송 없음: 개인정보를 외부 서버로 전송하지 않아 전송 구간 보안 위험 없음
  • 오픈소스: 소스 코드가 GitHub에 공개되어 있어 투명성 확보

10. 쿠키 및 추적 기술

본 확장 프로그램은:

  • 쿠키를 사용하지 않습니다.
  • 웹 비콘, 픽셀 태그 등 추적 기술을 사용하지 않습니다.
  • 사용자의 브라우징 활동을 추적하지 않습니다.
  • 분석 도구(Google Analytics 등)를 사용하지 않습니다.

11. 아동의 개인정보 보호

본 확장 프로그램은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 사용자가 개인정보를 입력한 경우, 법정대리인은 해당 정보의 열람, 정정, 삭제를 요청할 수 있습니다.

12. 개인정보처리방침의 변경

본 개인정보처리방침은 법령, 정책 또는 서비스 변경에 따라 수정될 수 있습니다.

  • 변경 시 본 페이지를 통해 공지합니다.
  • 중요한 변경 사항은 확장 프로그램 업데이트 시 알림을 통해 고지합니다.
  • 변경된 방침은 게시한 날로부터 효력이 발생합니다.
